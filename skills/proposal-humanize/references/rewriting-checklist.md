# Proposal Humanize Checklist

Use this checklist only when a proposal needs a deliberate second pass.

## 1. Structure

- Does the proposal reach the business point quickly?
- Is the sequence logical for the audience?
- Are sections uneven because one idea is repeated in multiple places?
- Would combining thin paragraphs improve authority?

## 2. Credibility

- Replace vague praise with operational meaning.
- Remove claims that sound inflated or unsupported.
- Keep commitments narrower than the evidence if needed.
- Make benefits sound earned, not declared.

## 3. Human Cadence

- Vary sentence openings.
- Mix short, medium, and long sentences.
- Remove repetitive connective phrases.
- Allow occasional plain language instead of constant polish.

## 4. Anti-AI Cleanup

Watch for patterns like:

- repeated abstract nouns
- symmetrical sentence templates
- stacked buzzwords
- generic scene-setting openings
- empty reassurance with no substance
- overuse of "comprehensive", "seamless", "robust", "tailored", "leverage", "enhance"

Replace with direct language tied to the actual proposal.

## 5. Style Transfer

When using a reference proposal:

- borrow structure, not wording
- borrow tone level, not signature phrases
- match evidence density
- match paragraph length only if it improves readability
- keep the client's facts and commercial position intact

## 6. Final Pass

- Read once for clarity.
- Read once for persuasion.
- Read once for anything that still sounds templated.
- End with a close that sounds confident but not over-scripted.
