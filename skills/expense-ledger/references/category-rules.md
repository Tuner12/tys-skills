# Category Rules

Use these categories unless the user asks for a different taxonomy.

## Categories

- `food`: meals, coffee, tea, snacks, groceries, delivery
- `transport`: taxi, uber, lyft, metro, bus, gas, parking, toll
- `shopping`: clothes, household items, general purchases
- `entertainment`: movies, games, concerts, leisure
- `bills`: phone bill, internet, utilities, insurance
- `housing`: rent, furniture, repair, maintenance
- `medical`: medicine, clinic, hospital, pharmacy
- `education`: books, course fees, tuition, software for study
- `travel`: hotel, flights, tickets, trip expenses
- `subscription`: recurring app or service fees
- `gift`: presents, donations, red envelope
- `digital`: domains, cloud, SaaS, software, AI tools
- `income`: salary, refund, reimbursement, bonus
- `other`: fallback when no strong category is detected

## Notes

- Prefer `food` over `shopping` for groceries and drinks.
- Prefer `medical` over `shopping` for medicine.
- Prefer `subscription` over `digital` when the charge is explicitly recurring.
- Prefer `travel` over `transport` when the expense is part of a larger trip.
